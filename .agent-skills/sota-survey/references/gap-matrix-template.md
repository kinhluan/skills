# SOTA Survey Gap Matrix Template

## Paper Entry Template

| Field | Description | Example |
|-------|-------------|---------|
| **Paper** | Full citation | Smith et al. (2024) |
| **Venue** | Conference/Journal | NeurIPS 2024 |
| **Problem** | What problem they solve | Privacy in FL |
| **Method** | Their approach | DP-SGD + compression |
| **Dataset** | Evaluation dataset | CIFAR-10, MNIST |
| **Metric** | Primary metric | Accuracy, Privacy budget ε |
| **Result** | Key result | 92.3% acc, ε=1.2 |
| **Limitation** | Known limitations | Only 10 clients tested |
| **CPY** | Citations Per Year | 15.2 |
